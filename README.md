# SecureTag

## Overview
AI-powered Secure Tag system designed to enhance cybersecurity and privacy through encrypted authentication and anomaly detection.

## Problem
Traditional RFID and digital tags are vulnerable to cloning, spoofing, and unauthorized access.

## Solution
Our system integrates encrypted secure tags with AI-based anomaly detection to prevent identity misuse and protect sensitive data.

## Tech Stack
RFID/NFC, ESP32, Python (Flask), MySQL, AES/RSA, JWT, Scikit-learn, Cloud Deployment.

## Status
Prototype development phase.
